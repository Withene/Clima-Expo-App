# Mini Projeto De Previsão do Clima!

### Funcionabilidade
* Exibição de clima atual, e previsão para 5 dias.
* Rastramento da cidade (Somente para previsão).
* Chance de chuva no local, Sensação termica, ou velocidade do vento.
